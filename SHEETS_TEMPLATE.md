## 컬럼 설명

- **id**: 숫자 (1, 2, 3, ...)
- **title**: 집단소송 제목
- **company**: 대상 기업명
- **status**: "모집중", "진행중", "완료" 중 하나
- **description**: 상세 설명
- **date**: YYYY.MM 형식 (예: 2025.11)
- **victims**: 피해자 규모
- **compensation**: 배상금 정보
- **category**: "소비자", "자동차", "금융", "가상자산", "IT·서비스", "증권" 등
- **link**: 참고 링크 URL

## 3단계: 시트 공개 설정

1. 우측 상단 "공유" 버튼 클릭
2. "일반 액세스" → "링크가 있는 모든 사용자" 선택
3. "뷰어" 권한으로 설정
4. 완료

## 4단계: Sheet ID 복사

URL에서 Sheet ID를 복사하세요:
```
https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
                                         ↑
                                    여기를 복사
```

## 5단계: 환경 변수 설정

`config.js` 파일에 Sheet ID를 입력하세요.

## 주의사항

- **첫 번째 행은 헤더**입니다 (컬럼명). 삭제하지 마세요!
- **Sheet 이름은 정확히 "lawsuits"**로 설정하세요
- status는 "모집중", "진행중", "완료" 중 하나만 사용
- date는 YYYY.MM 형식을 지켜주세요
- 새 소송을 추가하려면 새 행을 추가하면 됩니다
- 웹사이트는 **실시간으로 Sheets 데이터를 가져옵니다**
