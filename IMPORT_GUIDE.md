# Google Sheets CSV 임포트 가이드

## 📥 CSV 파일로 빠르게 시작하기

`lawsuits_sample.csv` 파일에는 현재 웹사이트의 모든 집단소송 데이터(10건)가 포함되어 있습니다.

## 방법 1: CSV 파일 직접 임포트 (가장 빠름)

### 1단계: 새 Google Sheets 생성
1. https://sheets.google.com 접속
2. 빈 스프레드시트 만들기

### 2단계: CSV 임포트
1. **파일** → **가져오기** 클릭
2. **업로드** 탭 선택
3. `lawsuits_sample.csv` 파일을 드래그 앤 드롭
4. 가져오기 위치: **데이터 바꾸기** 선택
5. 구분 기호 유형: **쉼표** 선택
6. **데이터 가져오기** 클릭

### 3단계: 시트 이름 변경
- 하단의 시트 이름을 **정확히 "lawsuits"**로 변경
- 우클릭 → 이름 바꾸기 → "lawsuits" 입력

### 4단계: 데이터 확인
- 첫 번째 행: 헤더 (id, title, company, ...)
- 두 번째 행부터: 10개의 집단소송 데이터

### 5단계: 시트 공개
1. 우측 상단 **"공유"** 버튼 클릭
2. **"일반 액세스"** → **"링크가 있는 모든 사용자"** 선택
3. 권한: **"뷰어"**로 설정
4. **"완료"** 클릭

### 6단계: Sheet ID 복사
URL에서 Sheet ID를 복사하세요:
```
https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
                                         ↑ 이 부분
```

예시:
```
https://docs.google.com/spreadsheets/d/1ABCdef-GHIjkl_MNOpqr/edit
                                         ↑
                                    Sheet ID
```

### 7단계: config.js 설정
프로젝트의 `config.js` 파일을 열고 Sheet ID를 입력:

```javascript
const SHEETS_CONFIG = {
    SHEET_ID: '1ABCdef-GHIjkl_MNOpqr',  // ← 여기에 복사한 Sheet ID 붙여넣기
    API_KEY: '',
    SHEET_NAME: 'lawsuits',
    // ...
};
```

### 8단계: 배포
```bash
git add config.js
git commit -m "Google Sheets 연동 설정"
git push
```

Vercel이 자동으로 재배포하고, 1-3분 후 웹사이트에 반영됩니다!

## 방법 2: 수동으로 데이터 입력

CSV 임포트 대신 직접 입력하려면:

### 컬럼 구조 (첫 번째 행)
```
id | title | company | status | description | date | victims | compensation | category | link
```

### 필드 설명

- **id**: 숫자 (1, 2, 3, ...)
- **title**: 집단소송 제목
- **company**: 대상 기업명
- **status**: "모집중", "진행중", "완료" 중 하나만 사용
- **description**: 상세 설명 (여러 줄 가능)
- **date**: YYYY.MM 형식 (예: 2025.11)
- **victims**: 피해자 규모 (예: 약 3,370만명)
- **compensation**: 배상금 정보 (예: 1인당 약 10만원 예상)
- **category**: "소비자", "자동차", "금융", "가상자산", "IT·서비스", "증권" 등
- **link**: 참고 링크 URL

## ✏️ 데이터 수정 및 추가

### 새 집단소송 추가
1. Google Sheets에서 새 행 추가
2. 데이터 입력 (위 필드 설명 참고)
3. 저장
4. **자동으로 웹사이트에 반영됩니다!** (최대 5분)

### 기존 소송 정보 수정
1. 해당 셀 클릭
2. 수정
3. 저장
4. **자동으로 반영!**

### 소송 삭제
1. 행 번호 우클릭
2. "행 삭제" 선택
3. **자동으로 반영!**

## 🔍 트러블슈팅

### Q: "데이터를 불러올 수 없습니다" 오류
**A:** 다음을 확인하세요:
- [ ] 시트 이름이 정확히 "lawsuits"인가?
- [ ] 시트가 "링크가 있는 모든 사용자"에게 공개되었나?
- [ ] Sheet ID가 config.js에 올바르게 입력되었나?

### Q: 웹사이트에 변경사항이 안 보여요
**A:**
- 캐시 때문에 최대 5분 걸릴 수 있습니다
- 브라우저 강력 새로고침: Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows)
- 개발자 도구 콘솔에서 에러 확인

### Q: CSV 임포트 시 한글이 깨져요
**A:**
- Google Sheets는 UTF-8을 자동으로 인식합니다
- 그래도 깨지면: 파일 → 가져오기 → 인코딩: UTF-8 선택

### Q: 숫자가 날짜로 변환돼요 (예: id가 날짜로)
**A:**
- 해당 열 선택 → 우클릭 → "일반 텍스트" 선택
- 또는 숫자 앞에 작은따옴표(') 추가: `'1`, `'2`, `'3`

## 📊 컬럼별 예시

### status (상태)
- `모집중` - 현재 참여자 모집 중
- `진행중` - 소송 진행 중
- `완료` - 소송 종결

### date (날짜)
- ✅ `2025.11`
- ✅ `2024.07`
- ❌ `2025-11-29` (잘못된 형식)
- ❌ `2025/11` (잘못된 형식)

### category (카테고리)
- `소비자` - 소비자 권리 관련
- `자동차` - 자동차 결함
- `금융` - 금융 사기, 펀드
- `가상자산` - 암호화폐, 코인
- `IT·서비스` - IT 서비스 장애
- `증권` - 주식, 증권 관련
